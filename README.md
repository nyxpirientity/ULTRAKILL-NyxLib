# About
NyxLib is *mostly* a library I created for my own personal ULTRAKILL mods- although it can technically be used by anyone, for any mods, I, particularly now, advise against it because I could change the API at anytime, and it's very possibly not even very good! Fundamentally, this doesn't matter much for people to read up on, it's just a framework for me to make mods for ULTRAKILL.

# Cheats
It features a few cheats, most exotic cheats are part of my *other* ULTRAKILL mods, but I'll list what there is in this one here alongside some explanation/description

## UTILITY:
- Radiant All Enemies - Does what it says- similar to forcerad, but it uses it's own configuration settings, configured in the config file. Why? Well, simply because the base ULTRAKILL radiance system was a bit too simplistic for my mods to combine from what I could tell, so I had to make a layer on top of it which simply takes in multiple *radiance modifiers* and compounds them together as either additives or multipliers. This *will* interfere with base game radiance if anything is actually using it, but if cheats are off, then as long as all mods are respecting my general policy of leaving all gameplay altering things behind cheats, it'll make no difference (in theory, but like, software has bugs and this is a big solo-unpaid-personal-project)
- Disable Stops - Disables hitstops
- Disable Slowdown - Disables slowdown
- Shortned Hitstop - halves hitstop time
- Always Play Battle Music - the battle never stops (for the music at least)
- Play Clean Music with Battle Music - Plays the calm/"clean" music alongside battle music

## SILLY:
- ULTRASTOP - (almost) all sound effects played cause hitstop (flashing lights warning!)
